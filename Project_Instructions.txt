Instruções para Configuração do Projeto ZebTrack-AI no GitHub
Visão Geral do Projeto
Este projeto visa unificar os repositórios PyZebArdYolo e ZebAdapt em uma única aplicação Python robusta e de código aberto para análise de comportamento de zebrafish, chamada ZebTrack-AI. A gestão será centralizada em um Projeto global que conecta os três repositórios para garantir a rastreabilidade.
1. Projeto Global (GitHub Project)
* Ação: Crie um novo "Projeto" no nível do usuário.
* Nome do Projeto: ZebTrack-AI Unification
* Template: Board (Kanban).
* Colunas: Backlog, To Do, In Progress, Done.
* Repositórios Vinculados: Adicione os seguintes repositórios ao projeto:
   1. marksant/ZebTrack-AI
   2. marksant/ZebAdapt
   3. marksant/PyZebArdYolo
