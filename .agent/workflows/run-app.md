# Run Application Workflow

// turbo-all

## Standard Launch

1. Run the GUI application:

```bash
poetry run zebtrack
```

## Development

1. Run with debug logging:

```bash
poetry run python -m zebtrack --log-level DEBUG
```

1. Run specific script:

```bash
poetry run python scripts/<script_name>.py
```
