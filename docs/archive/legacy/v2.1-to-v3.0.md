# Guia de Migração: v2.1 → v3.0

**Data de Release v3.0:** Fevereiro 2026
**Status:** Pré-lançamento (novembro 2025)

## 📋 Visão Geral

Esta versão remove APIs deprecadas introduzidas em v2.1 e introduz melhorias de performance.

### Breaking Changes

1. **Reporter Constructor Removal** (HIGH IMPACT)
2. **Settings Singleton Removal** (MEDIUM IMPACT - já migrado)
3. **EventBus API Changes** (LOW IMPACT)

---

## 🔴 1. Reporter Constructor Removal

### O que mudou?

O construtor direto do `Reporter` com `trajectory_df` foi **REMOVIDO**.

### Código Antigo (v2.1 - DEPRECATED)

```python
from zebtrack.analysis.reporter import Reporter

# ❌ NÃO FUNCIONA MAIS em v3.0
reporter = Reporter(
    trajectory_df=df,
    metadata={"experiment_id": "exp_001"},
    pixelcm_x=10.0,
    pixelcm_y=10.0,
    video_height_px=480,
    arena_polygon_px=polygon,
    rois=rois,
    fps=30.0,
    freezing_vel_threshold=1.0,
    freezing_min_duration=0.5,
)
```

### Código Novo (v3.0+)

```python
from zebtrack.analysis.analysis_service import AnalysisService
from zebtrack.analysis.reporter import Reporter
from zebtrack.settings import load_settings

# ✅ Caminho recomendado
settings = load_settings()
service = AnalysisService(settings_obj=settings)

analysis = service.run_full_analysis_as_dto(
    trajectory_df=df,
    pixelcm_x=10.0,
    pixelcm_y=10.0,
    video_height_px=480,
    arena_polygon_px=polygon,
    rois=rois,
    fps=30.0,
)

reporter = Reporter.from_analysis(analysis)
```

### Benefícios da Nova API

- ✅ **Performance:** Análise reutilizada, não recalculada
- ✅ **Type Safety:** DTO validado com Pydantic
- ✅ **Testabilidade:** Análise pode ser mockada
- ✅ **Separação de Concerns:** AnalysisService faz análise, Reporter faz reports

### Migration Script

Execute o script automático para migrar seus testes:

```bash
poetry run python scripts/migrate_reporter_v3.py --dry-run
poetry run python scripts/migrate_reporter_v3.py --apply
```

## 🟢 3. EventBus API Changes

### O que mudou?

`EventBus.publish()` agora requer `event_type` explícito.

### Código Antigo

```python
event_bus.publish("zone.draw_roi", data={"zone_id": 1})
```

### Código Novo

```python
from zebtrack.ui.events import EventType

event_bus.publish(EventType.ZONE_DRAW_ROI, data={"zone_id": 1})
```

**Impacto:** Baixo - afeta apenas código customizado que usa EventBus diretamente.
